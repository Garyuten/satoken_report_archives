レポートvol.9

  • 7/26(日) 14:00 ＠AIP Cafe

  • 参加人数:７人
九州／福岡は大雨警報が出てましたが、大分からわざわざ参加される方もいて、
少人数ではありましたが、いつも通りのサト研を開催できました。

少人数だったので、常にディスカッションし続ける形で、
いろんな話題にいろんな情報が派生していく感じのサト研でした。
（とてもぐでぐででした）

田村さん：Google Develpoer Day2009

  • HTML5
    • @kun_morita さんの質問「HTML5の対応のタイミングはどうする？」
    • Googleが推してる
    • GDDで公開されてるスライド「HTML5 により拓かれる次世代 Web 田村 健人」(PDF)が参考になる
      • フォームの強化
      • メディア要素
      • canvas
      • ドラッグ＆ドロップ
  • 対応ブラウザ
    • iPhone
    • Chrome (Android)
  • GDDで配ってたAndroid携帯見せてもらった
  • 個人的に感じたこと
    • Googleがやたらめったら力を入れていて、まだドラフト状態ながら早めに見てく必要がありそう
    • ベーシックなところは大きくは変わらなそうだけど、利用タグはそこそこ変わりそう
    • 動画や音声など主にインタラクティブなコンテンツ向けが手っ取り早く使えそう
    • 今からでも使えるが、基本というより、あるとより便利、よりおもしろいってことができｒ
    • Googleさんも主体に普及コミュができた模様
      • http://groups.google.co.jp/group/html5-developers-jp

やいぶさんのTIPSネタ

  • 木村カエラの壁紙
    • テンション上げるために重要
  • Finderウインドウ名にパスを表示
    • ターミナルで : defaults write com.apple.finder _FXShowPosixPathInTitle -bool YES
    • Finderを強制終了 : killall Finder
  • TextWrangler
    • テキスト比較
    • サーバーとローカルのファイル比較
    • チェックイン／チェックアウトの時に便利
  • お客さんの原稿やりとり
    • どうやって原稿をもらうか
    • EXCEL形式
      • シートにカンプを貼って、その上にテキストボックスを配置
        原稿のボリューム感を出せる
      • パワポだと1枚 1Pなので縦長いページなどには向いていない
